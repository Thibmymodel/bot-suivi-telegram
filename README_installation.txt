Installation du bot - Étapes essentielles

1. Placez votre fichier credentials.json (clé API Google) dans ce dossier.
2. Remplissez config.json avec vos informations (token, Google Sheet ID, etc.).
3. Installez les dépendances : pip install -r requirements.txt
4. Lancez le bot : python main.py
5. Pour Render, suivez le guide complet (fourni séparément si besoin).

Le bot gère : OCR des images Telegram, stockage dans Google Sheets, réponse Telegram avec résumé quotidien et hebdo.
